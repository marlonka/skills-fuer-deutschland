# Security Policy

Dieses Repository enthält Skill-Dateien und Dokumentation. Es enthält keine Anwendung, keinen Server und keine produktive Infrastruktur.

## Sicherheitsrelevante Meldungen

Bitte melde sicherheitsrelevante Probleme, wenn ein Beitrag:

- schädliche Anweisungen enthält;
- Nutzer zur Preisgabe sensibler Daten verleitet;
- vertrauliche Informationen, Zugangsdaten oder personenbezogene Daten enthält;
- gefährliche Rechts- oder Compliance-Scheinsicherheit erzeugt;
- Quellen oder Behördenentscheidungen frei erfindet.

Bitte eröffne dafür kein öffentliches Issue, wenn sensible Informationen enthalten sind. Nutze stattdessen die GitHub-Funktion für private Security Advisories, sofern verfügbar, oder kontaktiere den Repository-Inhaber direkt über GitHub.

## Umgang mit sensiblen Daten

Bitte keine echten Mandantendaten, personenbezogenen Daten, Vertragsgeheimnisse, Zugangsdaten, Aktenauszüge oder vertraulichen Dokumente in Pull Requests, Issues oder Beispielen verwenden.

Beispiele müssen anonymisiert oder fiktiv sein.
